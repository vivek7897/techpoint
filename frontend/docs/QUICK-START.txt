/*
   🚀 GETTING STARTED
   ==================
   
   1. Open 'codeseekho.html' in your web browser
   2. The platform will load with all features ready
   3. No installation or dependencies required!
   
   
   🎯 FIRST TIME SETUP
   ===================
   
   Step 1: Create an Account
   - Click "Sign Up" button in the top right
   - Enter your name, email, and password
   - Click "Sign Up" to create account
   - You'll be automatically logged in
   
   Step 2: Start Learning
   - Browse topics in the left sidebar
   - Click any topic to start learning
   - Read the content and examples
   - Your progress is automatically saved!
   
   
   ⚡ QUICK FEATURE TOUR
   ======================
   
   1. TUTORIAL CONTENT
      - Click topics in left sidebar
      - Read explanations
      - View code examples
      - Copy code with one click
   
   2. CODE PLAYGROUND
      - Edit HTML code in the editor
      - Click "Run Code" to see results
      - Click "Reset" to restore example
      - Click "Copy" to copy code
   
   3. DRAG & DROP BUILDER
      - Click the magic wand FAB button (bottom right)
      - Drag components from left panel
      - Drop them in the center canvas
      - See auto-generated HTML on the right
      - Click "Copy Code" to use it
   
   4. AI ASSISTANT
      - Click robot FAB button (bottom right)
      - Type your HTML question
      - Get instant answers
      - Use quick action buttons
   
   5. QUIZZES
      - Complete a topic to unlock quiz
      - Answer all questions
      - Click "Submit Answers"
      - See your score
      - Click "Next Topic" to continue
   
   6. DARK MODE
      - Click moon/sun icon in top right
      - Toggle between light and dark themes
      - Preference is automatically saved
   
   7. SEARCH
      - Use search bar in top navigation
      - Type topic name to filter
      - Results update as you type
   
   8. PROGRESS TRACKING
      - See progress bar in left sidebar
      - Shows completed/total topics
      - Updates automatically
      - Synced with your account
   
   9. CERTIFICATE
      - Complete all 18 topics
      - Certificate automatically appears
      - Download your achievement
      - Share your success!
   
   
   🔧 TROUBLESHOOTING
   ===================
   
   Issue: Nothing appears
   Fix: Make sure all 3 files are in same folder:
        - codeseekho.html
        - codeseekho-styles.css
        - codeseekho-script.js
   
   Issue: Progress not saving
   Fix: Check if browser allows localStorage
        Enable cookies and site data
   
   Issue: Can't login
   Fix: Make sure you signed up first
        Check email/password spelling
        Try signing up again
   
   Issue: Dark mode not working
   Fix: Click the moon icon in top right
        Refresh page if needed
   
   Issue: Code playground not working
   Fix: Click "Run Code" button
        Make sure HTML is valid
        Check browser console for errors
   
   
   💡 TIPS & TRICKS

   📱 MOBILE USAGE

   🎨 CUSTOMIZATION
   =================
   
   To customize colors:
   1. Open codeseekho-styles.css
   2. Edit :root variables:
      --primary: #04AA6D;
      --dark: #282A35;
      --light-bg: #f3f6f7;
      --white: #ffffff;
   
   To add topics:
   1. Open codeseekho-script.js
   2. Add to tutorialContent object
   3. Add to quizData object (optional)
